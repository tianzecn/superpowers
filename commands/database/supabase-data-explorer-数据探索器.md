# Supabase 数据探索器

使用智能查询和数据洞察探索和分析 Supabase 数据库：**$ARGUMENTS**

## 当前数据上下文

- Supabase MCP：以只读访问连接以安全探索数据
- 目标表：分析 $ARGUMENTS 以确定数据探索范围
- 本地查询：!`find . -name "*.sql" | head -5` 现有 SQL 文件供参考
- 数据模型：!`find . -name "types" -o -name "models" -type d | head -3` 应用程序数据结构

## 任务

执行综合数据库探索，包含智能分析和洞察：

**探索重点**：使用 $ARGUMENTS 指定表检查、SQL 查询执行、数据导出或全面数据库检查

**数据探索框架**：
1. **数据库发现** - 探索表结构、分析关系、识别数据模式、评估数据质量指标
2. **智能查询** - 通过 MCP 执行只读查询、优化查询性能、提供结果分析、建议查询改进
3. **数据分析** - 生成数据洞察、识别趋势和异常、计算统计摘要、分析数据分布
4. **架构检查** - 检查表架构、分析外键关系、评估索引效率、审查约束验证
5. **导出与可视化** - 以多种格式导出数据、创建数据可视化、生成摘要报告、优化数据呈现
6. **性能分析** - 分析查询执行计划、识别性能瓶颈、建议优化策略、监控资源使用

**高级功能**：交互式数据探索、自动化洞察生成、数据质量评估、关系映射、趋势分析。

**安全功能**：只读操作、查询验证、结果限制、性能监控、错误处理。

**输出**：全面的数据探索，包含洞察、优化查询、导出文件和性能建议。
